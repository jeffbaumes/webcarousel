Web Carousel

Settings

Information

Web Carousel can turn any browser-capable display into a carousel flipping through web pages. Some ideas:
  • A Google calendar, by going to Calendar Settings then Embed This Calendar. Note: Use just the link in the embed code (URL inside src="..."), not the entire iframe code.
  • A weather page from Weather Underground.
  • A realtime stock price page.
  • Note that some pages that do not like being embedded (e.g. Gmail or Google Search) will not work with this app.
Settings will be saved to your local browser. No information is stored on the application server.
